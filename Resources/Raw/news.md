# IndoorCO2Map News

## Welcome

Welcome to the IndoorCO2Map New App Update! This is still very much a early beta Version.
This page will contain important news regarding the App and Project and sometimes also general things concerning indoor air quality.

### New Features

You can now sort by distance or alphabetically and there is also a search option for the location results
There is a history page which shows your own recordings. They can be exported (as SQlite Database for now) and imported again, so transfering to other devices while keeping your own recording dataset is possible. This history is only stored locally on your phone, but nonetheless this is an optional feature you can disable in the Settings Menu.
Some other features which make the app more pleasant to use


### Limitations of the Beta Version
most importantly the transit mode is not implemented yet. The old implementation was very rudimentary and for this update I have a lot of ideas to improve it but these take time and will require setting up a Server with the Open Street Map Data for ourselves rather than relying on public available free servers.
Currently the new App is only available in english - the release version will be multi-language as before.
Only Aranet and Airvalent supported for now



### Visual Overhaul

The App got some visual improvements. Please let me know how you like it

### Statistics Page

The IndoorCO2Map database keeps growing — thank you to everyone contributing measurements!
Head over to the *Statistics* page to see the latest breakdown by country. This will eventually contain your own data for comparison as well and way more different statistics
